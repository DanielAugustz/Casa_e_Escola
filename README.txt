Projeto CASA E ESCOLA


Página Inicial:
Na página inicial, é possível visualizar um menu, o logotipo do projeto e um informativo com uma imagem de fundo. Basta clicar em uma das opções do menu, no botão "Saiba mais" ou até mesmo usar a barra de rolagem.
  


Página Sobre:
Ao clicar em “Sobre” no menu ou no botão “Saiba Mais”, você será redirecionado para esta página que apresenta informações sobre o projeto. Para sair dela, basta usar o botão de rolagem do mouse.
  


Página Professores:
Ao clicar em “Professores” no menu, você será redirecionado para esta página, que contém um informativo sobre as futuras funcionalidades que os professores poderão utilizar no site do projeto. Para sair dela, basta usar o botão de rolagem do mouse.
  

Página Aluno:
Ao clicar em “Aluno” no menu, você será redirecionado para esta página que contém um informativo sobre as futuras funcionalidades que o aluno poderá usar no site do projeto. Para sair dela, basta usar o botão de rolagem do mouse.
  

Página Contatos:
Ao clicar em “Contato” no menu, você será redirecionado para esta página, que oferece um meio de entrar em contato conosco. Basta preencher as informações necessárias e enviar. Além disso, há outras formas de contato disponíveis no rodapé do site, que está destacado em azul. Para sair da página, basta usar o botão de rolagem do mouse.
